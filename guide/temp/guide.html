컴포넌트 가이드

프로젝트에서 공통으로 사용하는 UI 패턴과 마크업 기준을 정리한 샘플 페이지입니다.

Button

기본 버튼

주요 액션과 보조 액션에 사용하는 버튼입니다.

<button type="button" class="btn">Primary</button>
<button type="button" class="btn sub">Secondary</button>

Input

텍스트 입력

폼 영역에서 공통으로 사용하는 인풋 샘플입니다.

<div class="field">
  <label for="name" class="label">이름</label>
  <input
    type="text"
    id="name"
    class="input"
    placeholder="이름을 입력하세요"
  />
</div>

Card

정보 카드

목록형 UI에서 재사용 가능한 카드 예시입니다.

카드 제목 1

카드에 들어가는 설명 텍스트 예시입니다.

카드 제목 2

간단한 소개 문구나 상태값을 넣을 수 있습니다.

카드 제목 3

리스트형 콘텐츠를 정리할 때 사용합니다.

<div class="item">
  <strong>카드 제목</strong>
  <p>카드에 들어가는 설명 텍스트입니다.</p>
</div>
